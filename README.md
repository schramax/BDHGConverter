# BinDecHexGray Converter

A real-time, responsive, and installable Progressive Web App (PWA) for converting unsigned integers between Normal and Gray-code representations. 

App URL: [BDHG Converter App](https://schramax.github.io/BDHGConverter/)

## Features

- **Real-Time Conversion**: Instantly converts between Binary, Decimal, and Hexadecimal representations for both Normal and Gray-code formats as you type.
- **Arbitrary Precision**: Built using JavaScript's `BigInt`, allowing you to safely calculate and manipulate massive bitwidths without precision loss.
- **Smart Formatting**: Binary fields and Hexadecimal fields automatically format with spaces for optimal readability. 
- **Dynamic Bitwidth**: Easily scale up or down your bitwidth limit. The application intelligently truncates and masks values based on your configuration.
- **Progressive Web App (PWA)**: Completely installable on desktop and mobile devices. Works 100% offline via Service Worker caching.
- **Modern UI**: Features a beautiful dark mode, glassmorphism aesthetics, and keyboard-friendly up/down input spinners.
